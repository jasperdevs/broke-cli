Routing

Cheap turns stay cheap.

brokecli routes easy requests to smaller models, preserves stronger models for the turns that actually need them, and makes the cost trade visible instead of magical.

Auto-route. Smaller lane for low-complexity work.
Budget telemetry. Track wasted tool and token spend.
Scoped models. Pin preferred models for quick cycling.

Workflow

One command. Clear signal.

Static page outside. sharp runtime inside. Start with a prompt, keep the transcript, and let the CLI handle compacting, routing, exporting, and validation without leaving the terminal. 

# install
npm install -g @jasperdevs/brokecli

# interactive coding
brokecli

# one-shot / rpc
brokecli -p "summarize this diff"
brokecli --rpc
Why it sticks

Less gloss. More leverage.

Most AI coding products optimize for spectacle. brokecli optimizes for the second week, when context is long, sessions are messy, and somebody has to justify the bill.

  • Keep history usable. Resume, fork, compact, export, and share without ritual.
  • Keep cost visible. Small turns route down; expensive turns stay intentional.
  • Keep control local. Hosted models, local models, extensions, and permissions coexist.
